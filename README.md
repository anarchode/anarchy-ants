# Anarchy Ants
What if Langton's ant could turn whatever angle it wanted?
Well, this is that.

## Requirements
python2.7
pygame

## Examples

### Classic Langton's ant
Ant(1,90)

![90 degrees](examples/90degrees.png)

### 5 30 degree ants
Ant(5,30,True)

![30 degrees](examples/30degrees.png)

### 90.2 degree ant
Ant(1,90.2)

![90.2 degrees](examples/90.2degrees.png)
